二叉树、图

二叉树的种类：

  • 满二叉树：深度为k，有2^k-1个节点的二叉树，即除了最下一层的节点度为0，其余节点度都为2
  • 完全二叉树：只可能最底层的节点没填满，且最底层的节点一定都集中在左边

    堆是完全二叉树和排序的结合，父节点一定≥子节点，它不是平衡二叉搜索树。

  • 二叉搜索树：若左子树不空，则左子树上的所有节点上的值都小于它根节点；若右子树不空，则右子树上的所有节点上的值都大于它根节点的值
  • 平衡二叉搜索树（AVL）：每一个节点的左右子树的高度差不超过1

    「C++中 map、set、multimap，multiset 的底层实现都是平衡二叉搜索树，也即红黑树」，所以map、set的增删操作时间时间复杂度是lognlog nlogn﻿，注意unordered_map、unordered_map底层实现是哈希表。

二叉树的存储：

  • 链式存储
  • 顺序存储：如果一个节点的下标是i，则它的左孩子是2*i+1，右孩子是2*i+2

遍历二叉树的时间复杂度为O(n)

    递归相当于栈

  • 广度优先遍历

递归的三步法：

  1. 确定递归函数的参数和返回值
  1. 确定终止条件

    例如节点为空则返回

  1. 确定单层递归的逻辑

    例如前序遍历：先把当前节点值放入vector，再递归左节点，再递归右节点

迭代方法：

按照递归走，要把终止条件（返回条件）加进去，加入vec的地方也是跟递归一致的。后序遍历复杂一些。

//以下是迭代方法
//前序遍历
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> result;
    stack<TreeNode*> stk;
    TreeNode* node = root;
    while (!stk.empty() || node!=nullptr){
        while (node!=nullptr){
            result.emplace_back(node->val);
            stk.emplace(node);
            node = node->left;
        }
        node = stk.top();
        stk.pop();
        node = node->right;
    }
    
    return result;
}
//后序遍历
vector<int> postorderTraversal(TreeNode* root) {
    vector<int> result;
    stack<TreeNode*> stk;
    TreeNode* node = root;
    TreeNode* prev = nullptr;
    while (!stk.empty() || node!=nullptr){
        while (node!=nullptr){
            stk.emplace(node);
            node = node->left;
        }
        node = stk.top();
        stk.pop();
				//需要用prev来记录，右子树是不是已经遍历过了
        if (node->right==nullptr || node->right==prev){
            result.emplace_back(node->val);
            prev = node;
						//这里置为nullptr是告诉下一次迭代，左子树已经遍历过了
            node = nullptr;
        }
        else{
            stk.emplace(node);
            node = node->right;
        }
    }
    return result;
}

递归法返回的数据类型，如果涉及深度、高度，一般为int，如果要判断是否平衡，可以用返回-1表示不平衡。

递归函数什么时候要有返回值？

  • 「如果需要遍历整颗树，递归函数就不能有返回值。如果需要遍历某一条固定路线，递归函数就一定要有返回值！还有一种就是后序遍历需要根据左右递归的返回值推出中间节点的状态，这种需要有返回值。」
  • 可以通过返回TreeNode*节点，向树添加新节点
  • 求高度传入节点就行，返回当前节点的高度，然后上一层节点累加上去；求深度返回为空，传入参数要有当前深度，用全局变量记录最大的深度。

回溯是一种复杂的迭代？

用迭代法求数的深度、节点个数的时候，要注意边界条件（返回条件）是只看那一个节点就行，例如节点为空，则返回深度为0，或返回节点个数为0。

二叉搜索树总结：

  • 大多数题目都可以用中序遍历将一个二叉搜索树变成一个数组，再进行处理。
  • 如果可以用有序数组做，就可以用递归遍历，可以在递归遍历的时候用pre指针指向前一个节点，一边遍历一边处理。

删除节点的方法：

  • 如果是普通二叉树，要删除某一个节点，把它的值跟左/右子树某个叶子节点的值交换，然后再次遍历到该值时，删除即可。
  • 如果是二叉搜索树，两种方法
    • 把它的值跟右子树最左边节点的值交换，再次遍历到该值时，就是右子树最左边的节点了，此时如果右子树为空，直接返回左子树，如果右子树不为空，再次跟右子树最左边节点交换。。。如此往复。
    • 把它左子树放到右子树最左边节点的左孩子处，返回右子树 / 或者把右子树放到左子树最右边节点的右孩子处，返回左子树 即可

注意点：

  • 如果提到叶子结点，判断一定要是 node->left==nullptr && node->right==nullptr ，而不能是 node==nullptr 。
  • 关于二叉树中的指针，如果是指针指向的值，如 node->val node->left node->right ，可以直接传入指针即可修改，如果是指针本身的值，如 *node1 = *node2 ，令指针1指向指针2，则要传入指针的指针。
  • 对于要从底向上搜索的问题，要用后序遍历。
  • 可以通过返回TreeNode*节点，向树添加新节点，或者删除节点
  • 二叉树的题目，多想想递归，少想想像排列组合那样的回溯

统计最高出现频率的元素的集合的方法：

  • 对于非排序数组，

    用map统计每个元素出现次数，key是元素，value是出现次数，然后将map对出现次数进行排序，这个排序可以用sort函数，但是第三个参数比较方式的函数对象要自己定义。排好之后，取第一个元素或前n个元素即可。

  • 对于排序数组，

    先遍历一遍寻找最大频率，再遍历一遍加入该频率的元素们，如果不是集合（只需要一个），则遍历一遍就够了。
