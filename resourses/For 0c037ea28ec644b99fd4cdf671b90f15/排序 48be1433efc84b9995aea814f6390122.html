排序

快速排序

基本思想：

  1. 先从数列中取一个数作为基准
  1. 把大于它的数都放到它的右边，小于它的数都放到它的左边
  1. 再对左右区间重复上述步骤，直到各区间只有一个数

挖坑填数+分治法

复杂度分析：

  • 时间复杂度：假如每次都能把数列分成长度相等的两半，那么需要递归 log2Nlog_2Nlog2​N﻿ 层，每一层需要对整个数列遍历一次，即 O(N)O(N)O(N)﻿ ，因此总的时间复杂度为 O(Nlog2N)O(Nlog_2N)O(Nlog2​N)﻿ 。每一层的O(N)可能难以理解，例如N分成两半，每一半遍历N/2遍，这两半在同一层，加起来就是N遍。
  • 空间复杂度：应该就是递归时候的递归栈吧，O(log2N)O(log_2N)O(log2​N)﻿
void QuickSort(vector<int>& array, int first, int last){
    if (first >= last)
        return;
    
    int l = first, r = last;
    int key = array[l];
    while (l < r){
        while (l < r && array[r] >= key){
            r--;
        }
        array[l] = array[r];
        while (l < r && array[l] <= key){
            l++;
        }
        array[r] = array[l];
    }
    array[l] = key;
    
    QuickSort(array, first, l-1);
    QuickSort(array, l+1, last);
}

tips：感觉快速排序适合用闭区间做。如果要取中间的值作为key的话，可以先跟首或尾交换。

记录一下单向链表的快速排序：

class Solution {
public:
    ListNode* sortList(ListNode* head) {
        return quickSort(head,nullptr);
    }
    ListNode* quickSort(ListNode* head, ListNode* tail){
        if (head==tail || head->next==tail){
            return head;
        }
        int base = head->val;
        ListNode* dummy = new ListNode(-1);
        dummy->next = head;
        ListNode* node = head;
        while (node->next!=tail){
						//如果当前节点值小于base值，插到开头，就不用往后移动一个了
            if (node->next->val < base){
                ListNode* cur = node->next;
                node->next = cur->next;
                cur->next = dummy->next;
                dummy->next = cur;
            }
						//否则往后移一个节点
            else{
                node = node->next;
            }
        }
				//注意这里要将dummy的下一个连接到其返回值
        dummy->next = quickSort(dummy->next, head);
        head->next = quickSort(head->next, tail);
        return dummy->next;
    }
};

归并排序

基本思想：

利用分治思想，将一个数列分成两半，再分两半，……，直到只剩一个数，再回上一层，将这两个数按大小顺序合并，再返回上一层，将已经排好序的两半按大小顺序合并，……，直到第一层，就可以得到排好的数列。

其中每一层合并的时候，要用额外的空间，先放在额外空间中，再复制到原数组中。

复杂度分析：

  • 时间复杂度：递归层数为O(log2N)O(log_2N)O(log2​N)﻿，每一层相当于对每个数进行了遍历O(N)O(N)O(N)﻿，所以时间复杂度为O(Nlog2N)O(Nlog_2N)O(Nlog2​N)﻿
  • 空间复杂度：一个额外的N大小的数组用于归并时临时存放，所以空间复杂度为O(N)O(N)O(N)﻿
void recursive(vector<int>& array, int start, int end, vector<int>& tmp){
    if (start >= end)
        return;
    int mid = (start + end) / 2;
    recursive(array, start, mid, tmp);
    recursive(array, mid+1, end, tmp);
    int first = start, second = mid+1;
    int k = start;
    while (first <= mid && second <= end){
        if (array[first] <= array[second]){
            tmp[k++] = array[first++];
        }
        else{
            tmp[k++] = array[second++];
        }
    }
    while (first <= mid){
        tmp[k++] = array[first++];
    }
    while (second <= end){
        tmp[k++] = array[second++];
    }
    for (int i=start; i<=end; ++i){
        array[i] = tmp[i];
    }
}

注：如果采用链表而不是数组，可以不需要开辟O(N)的额外空间，因为数组merge的时候会把原数组覆盖掉无法继续比较，而链表只需要交换，不会覆盖

堆排序

基本思想：

构建一个大顶堆（升序）或小顶堆（降序），对于大顶堆，每个节点都大于等于两个子节点的值。构造好之后，再进行节点下沉，每次把最顶端的数与最后一个数交换（未排序的最后一个数），再调整堆，如此往复，最后形成的完全二叉树就是按序的了。

复杂度分析：

  • 时间复杂度：假设每个节点要O(logN)O(logN)O(logN)﻿来调整，总O(NlogN)O(NlogN)O(NlogN)﻿
  • 空间复杂度：O(1)O(1)O(1)﻿
void maxheap(vector<int>& array, int start, int end){
    int k = 2*start+1;
    while (k < end){
        if (k+1<end && array[k+1]>array[k]){
            k++;
        }
        if (array[start] < array[k]){
            swap(array[start], array[k]);
        }
        start = k;
        k = 2*start+1;
    }
}

void heap_sort(vector<int>& array){
    int n = array.size();
    for (int i=(n-2)/2; i>=0; --i){
        maxheap(array, i, n);
    }
    for (int i=n-1; i>=0; --i){
        swap(array[0], array[i]);
        maxheap(array, 0, i);
    }
}

插入排序

基本思想：

从前往后，每次取出一个数，插入到前面已经排好序的数列中。

具体实现在原地实现，在前面排好的数列中找到这个数应该在的位置，把后面的数都往后移一个，把这个数放进来。

复杂度分析：

  • 时间复杂度：最好情况是本身已经有序，那就只需要O(N)O(N)O(N)﻿的复杂度；最坏情况是本身是逆序的，那么每一个数都要遍历两次，则需要O(N2)O(N^2)O(N2)﻿的复杂度——是0+1+...+N−1=12N(N−1)0+1+...+N-1=\frac{1}{2}N(N-1)0+1+...+N−1=21​N(N−1)﻿算出来的？
  • 空间复杂度：O(1)O(1)O(1)﻿
void InsertSort(vector<int>& nums){
    for (int i=1; i<nums.size(); ++i){
        int cur = nums[i];
        int j = i - 1;
        for (; j>=0 && nums[j]>cur; --j){
            nums[j+1] = nums[j];
        }
        nums[j+1] = cur;
    }
}

希尔排序

基本思想：

是插入排序的改进版。它通过比较相距一定间隔的元素来进行，各趟比较所用的距离随着算法的进行而减小，直到只比较相邻元素的最后一趟排序为止。一般间隔从length开始每次除以二。

底层用的插入排序，但是复杂度优于插入排序，因为插入排序对于相对有序的数列性能较好，希尔排序就通过使它相对有序优化它的性能。

从另一个角度来说，排序的本质是消灭逆序，如果一次交换相邻元素，只能消灭一个逆序，而希尔排序通过交换相隔较远的数一次消灭多个逆序，快排、堆排等等算法都是交换比较远的元素，只不过规则各不同罢了。

复杂度分析：

  • 时间复杂度：O(N1.3−2)O(N^{1.3-2})O(N1.3−2)﻿
  • 空间复杂度：O(1)O(1)O(1)﻿
void ShellSort(vector<int>& array){
    int n = array.size();
    for (int gap=n/2; gap>=1; gap/=2){
        for (int i=0; i<gap; ++i){
            for (int j=i+gap; j<n; j+=gap){
                int tmp = array[j];
                int k = j-gap;
                for (; k>=0 && array[k]>tmp; k-=gap){
                    array[k+gap] = array[k];
                }
                array[k+gap] = tmp;
            }
        }
    }
}

冒泡排序

基本思想：

从前往后两两比较，如果左>右，则交换，这样每次会把最大的数换到最右边。类似于大的泡泡会往上冒得最快。

复杂度分析：

  • 时间复杂度：要进行N轮排序，N-1+...+2+1，所以时间复杂度为O(N2)O(N^2)O(N2)﻿。
  • 空间复杂度：没有用额外的存储空间，O(1)O(1)O(1)﻿。
void BubbleSort(vector<int>& array){
    //增加标志位来优化，如果这一次循环中没有交换，则表明已经有序
    bool swapped;
    for (int i=0; i<array.size(); ++i){
        swapped = false;
        for (int j=0; j<array.size()-1-i; ++j){
            if (array[j] > array[j+1]){
                swap(array[j], array[j+1]);
                swapped = true;
            }
        }
        if (!swapped){
            break;
        }
    }
}

选择排序

基本思想：

从前往后遍历，每一趟都找到剩余数列中最小的数，与剩余数列的第一个数交换。这样每次会把最小的数换到最前面。

复杂度分析：

  • 时间复杂度：要进行N轮排序，N+N-1+...+2+1，所以时间复杂度为O(N2)O(N^2)O(N2)﻿。
  • 空间复杂度：O(1)O(1)O(1)﻿
void SelectSort(vector<int>& array){
    for (int i=0; i<array.size(); ++i){
        int minind = i;
        for (int j=i+1; j<array.size(); ++j){
            if (array[j] < array[minind]){
                minind = j;
            }
        }
        swap(array[i], array[minind]);
    }
}

注意选择排序的不稳定性：例如5 8 5 2 9，会把第一个5换到2的位置，在第二个5后面，后续遍历会先把第二个5换上去

计数排序

基本思想：

开辟一块k大小的空间，k是数列中最大数-最小数+1，每个数映射到k个块中的某一个。

  1. 遍历原数组计数，放入count数组
  1. 遍历count数组，计算每个位置及≤它有的个数
  1. 遍历原数组，可以映射到count数组知道它排在第几个，放到一个新数组中，并把该位置的计数-1（应对有重复元素的情况）
  1. 把新数组复制到原数组

感觉2、3步就是为了稳定性，即相等元素中，原本在前的依旧在前。

复杂度分析：

  • 时间复杂度：O(N+k)O(N+k)O(N+k)﻿，k是count数组的大小。
  • 空间复杂度：O(N+k)O(N+k)O(N+k)﻿
void CountSort(vector<int>& array){
    int n = array.size();
    if (n == 0)
        return;
    int max = array[0];
    int min = array[0];
    for (int i=0; i<n; ++i){
        max = array[i]>max ? array[i] : max;
        min = array[i]<min ? array[i] : min;
    }
    int size = max-min+1;
    vector<int> count(size,0);
    for (int i=0; i<n; ++i){
        count[array[i]-min]++;
    }
    for (int i=1; i<size; ++i){
        count[i] += count[i-1];
    }
    vector<int> psort(n);
    for (int i=n-1; i>=0; --i){
        count[array[i]-min]--;
        psort[count[array[i]-min]] = array[i];
    }
    for (int i=0; i<n; ++i){
        array[i] = psort[i];
    }
}

桶排序

基本思想：

桶排序是计数排序的扩展版本，计数排序可以看成每个桶只存储相同元素，而桶排序每个桶存储一定范围的元素，通过映射函数，将待排序数组中的元素映射到各个对应的桶中，对每个桶中的元素进行排序，最后将非空桶中的元素逐个放入原序列中。

复杂度分析：

  • 时间复杂度：O(N+k)O(N+k)O(N+k)﻿
  • 空间复杂度：O(N+k)O(N+k)O(N+k)﻿
void bucket_sort(vector<int>& array){
    int n = array.size();
    if (n == 0)
        return;
    int minnum = array[0];
    int maxnum = array[0];
    for (int i=0; i<n; ++i){
        if (array[i] < minnum)
            minnum = array[i];
        if (array[i] > maxnum)
            maxnum = array[i];
    }
    
		//将每个桶的可放数字范围为10
    int bucket_len = 10;
    int nbucket = (maxnum-minnum)/bucket_len + 1;
    vector<vector<int>> bucketlist(nbucket,vector<int>());
    
		//将数入桶
    for (int i=0; i<n; ++i){
        int num = (array[i]-minnum)/bucket_len;
        bucketlist[num].push_back(array[i]);
    }
    
		//桶内排序
    for (int i=0; i<nbucket; ++i){
        vector<int> nums = bucketlist[i];
        for (int j=1; j<nums.size(); ++j){
            int tmp = nums[j];
            int k = j-1;
            for (; k>=0 && nums[k]>tmp; --k){
                nums[k+1] = nums[k];
            }
            nums[k+1] = tmp;
        }
    }

    //把桶中元素依次放回原数组
    int index = 0;
    for (int i=0; i<nbucket; ++i){
        for (int j=0; j<bucketlist[i].size(); ++j){
            array[index] = bucketlist[i][j];
            index++;
        }
    }
    
}

基数排序

基本思想：

将整数按位数切割成不同的数字，然后按每个位数分别比较。

复杂度分析：

  • 时间复杂度：O(d∗(N+k))O(d*(N+k))O(d∗(N+k))﻿，其中，d是位数，k是基数，例如这里是10
  • 空间复杂度：O(N+k)O(N+k)O(N+k)﻿
int getbitnum(vector<int>& array){
    int maxnum = array[0];
    for (int num : array){
        if (num > maxnum)
            maxnum = num;
    }
    int nbit = 0;
    while (maxnum > 0){
        nbit++;
        maxnum /= 10;
    }
    return nbit;
}

void RedixSort(vector<int>& array){
    int nbit = getbitnum(array);
    
    int redix = 1;
    for (int i=0; i<nbit; ++i){
        vector<vector<int>> buckets(10,vector<int>());
        for (int j=0; j<array.size(); ++j){
            int num = array[j];
            int nbucket = num/redix % 10;
            buckets[nbucket].push_back(num);
        }
        int index = 0;
        for (int k=0; k<10; ++k){
            for (int num : buckets[k]){
                array[index] = num;
                index++;
            }
        }
        redix *= 10;
    }
}

注：计数排序、桶排序、基数排序都是非比较型排序，都利用了桶的概念，但有差异：

  • 基数排序：根据键值的每位数字来分配桶
  • 计数排序：每个桶只存储单一键值
  • 桶排序：每个桶存储一定范围的数值
