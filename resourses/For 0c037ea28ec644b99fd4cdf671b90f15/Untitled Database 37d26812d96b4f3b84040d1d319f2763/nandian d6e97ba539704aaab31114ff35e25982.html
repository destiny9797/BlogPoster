1、头文件相互引用的问题

不同的类定义在不同的头文件中，而不同的类之间又都用到了对方类。

解决：使用类指针而不是类对象，可以不用包含头文件，在使用前有简单的声明即可。也不能用类中定义的sptr，应该在外部typedef。

2、如果要未实例化类对象就用类中的成员，成员必须定义为static

3、boost::mutex在函数中必须以指针或引用传递，因为singleton？

4、extern的变量，除了在头文件中声明，还要在源文件中定义。

5、In C++, all "virtual parents" are always "direct". There are no "virtual grandparents", which is what I think you're doing. That's how virtual inheritance works. Virtually inherited classes are always the parent's of the most-derived class, which directly inherits them, whether explicitly declared as such, or not. C++ does not work any other way.

6、循环缓存是怎么实现的？

通过mmap映射，将两段连续的内存映射到同一个文件中，这样两段内存对应位置应该指向同一个文件的同一个位置，则数据相同。

具体做法：先映射2*size大小的内存，然后将后半部分取消映射，再从first_copy+size处映射second_copy，映射大小为size，再将文件截断，只留前半部分。
